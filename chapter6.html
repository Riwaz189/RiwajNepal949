History of Internet

The internet began as a research project in the 1960s called ARPANET, funded by the U.S. Department of Defense. It was initially developed to connect different computers for academic and military use. Over time, it evolved into a vast global communication network. The introduction of the World Wide Web in 1991 by Tim Berners-Lee made the internet accessible and user-friendly. This revolutionary invention transformed communication, commerce, education, and entertainment, making the internet an essential part of modern life.

Terminologies in Web Technologies

Web technologies include a variety of components that enable communication and content sharing over the internet. A webpage is a single document on the web, while a website is a collection of related webpages. A web browser is software that allows users to access webpages, such as Chrome or Firefox. Servers are powerful computers that host websites and respond to requests from clients (browsers). URL stands for Uniform Resource Locator and specifies the address of a resource. HTTP/HTTPS are protocols used to transfer data between client and server.

HTML (HyperText Markup Language)

HTML is the standard markup language used to create the structure of webpages. It defines how content is organized using elements like headings, paragraphs, images, and links. Each element is represented by tags enclosed in angle brackets. HTML forms the backbone of any website and ensures that the content is correctly displayed in web browsers. Tags in HTML

HTML tags are predefined keywords used to mark up content in an HTML document. They usually come in pairs: an opening tag (e.g., <p>) and a closing tag (e.g., </p>). Common tags include <h1> to <h6> for headings, <a> for hyperlinks, <img> for images, <ul> and <li> for lists, and <div> for division of content. Tags help browsers understand how to display the content. Understanding the correct usage of tags is fundamental to creating functional and accessible webpages.

CSS (Cascading Style Sheets)

CSS is a style sheet language used to describe the look and formatting of a document written in HTML. While HTML structures content, CSS defines the visual appearance like colors, fonts, layout, and spacing. CSS enhances the user experience by making web interfaces more attractive and responsive. It allows developers to separate design from content, promoting cleaner code and easier maintenance. CSS can be applied in various ways to suit different needs.

Types of CSS

There are three main types of CSS:
Inline CSS: Applied directly within an HTML tag using the style attribute. Useful for small changes.
Internal CSS: Defined within a <style> element inside the <head> section. Affects only the current page.
External CSS: Written in a separate .css file and linked to HTML pages. Best for consistent styling across multiple pages. These methods allow developers flexibility in styling webpages according to project requirements.
